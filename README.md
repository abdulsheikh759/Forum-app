Mini Community Forum (Full Stack Project)

This project is a Mini Community Forum application built as part of a junior-level technical assignment.
Users can create groups, join them, post messages, like posts, and interact using comments.

The focus of this project is functionality, API design, and backend logic, not UI perfection.


Tech Stack
Backend
Node.js
Express.js
MongoDB (Mongoose)
JWT Authentication

Frontend
React
React Router DOM
Context API
Axios
Tailwind CSS (minimal styling)


Core Features
User signup & login (JWT based)
Create and join groups
Create posts inside groups
Like / Unlike posts
Add and delete comments
Protected routes (only authenticated users)
RESTful API architecture
